# Screening Answers

Reusable answers to common screening questions. The `tailor-application`
skill draws from here when drafting `answers.md` for a posting.

## Format

Each entry is an H3 question followed by the answer, then tags.

```markdown
### Why are you looking for a new role?

<answer text>

`tag:motivation` `tag:short`
```

Tags are free-form but should at minimum signal topic and length
(`tag:short`, `tag:medium`, `tag:long`).
