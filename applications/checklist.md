# Application Checklist

Copy this checklist for each new application.

---

## Template: [Company Name] — [Role]

### Before Applying
- [ ] Research the company (mission, products, culture, recent news)
- [ ] Research the team/department if possible (LinkedIn, blog posts)
- [ ] Identify 2-3 specific reasons you want this role at this company
- [ ] Run `python tools/tailor_resume.py --job "jd.txt" --company "X" --role "Y"` for tailoring suggestions
- [ ] Update resume with suggested keywords and bullet rewrites
- [ ] Run `python tools/generate_cover_letter.py --job "jd.txt" --company "X" --role "Y"` to generate cover letter
- [ ] Review and personalize the AI-generated cover letter
- [ ] Save job description to `applications/jds/company_role.txt`
- [ ] Add entry to `tracker.md`

### Submitting
- [ ] Final resume review (no typos, correct dates, consistent formatting)
- [ ] Final cover letter review (correct company name, no leftover template text)
- [ ] Note the application deadline
- [ ] Submit application
- [ ] Update tracker status to ✉️ Applied
- [ ] Set follow-up reminder (7-10 days)

### After Applying
- [ ] Connect with someone at the company on LinkedIn (optional)
- [ ] Send follow-up email if no response after 7-10 days
- [ ] Update tracker with any responses

### If You Get a Screen/Interview
- [ ] Confirm date, time, format (phone/video/on-site), and interviewer name
- [ ] Review company research notes
- [ ] Prep STAR stories in `interview-prep/notes.md`
- [ ] Prepare questions to ask the interviewer
- [ ] Send thank-you note within 24 hours of interview
